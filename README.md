# bcche-ui-react

## Getting Started

Install dependencies,

```bash
$ npm i
```

开发环境：

```bash
$ npm run dev
```

生成文档
```bash
$ npm run docs:build
```

测试

```bash
$ npm test
```

修改版本号，生成npm包

```bash
$ npm run build
```
提交代码到远程main分支，更新npm包

```bash
$ npm publish 
```

git push到main分支之后，页面会自动部署
